## Run Locally

Clone the project

```bash
  git clone https://github.com/SoapDev2018/stashscript
```

Go to the project directory

```bash
  cd stashscript
```

Install dependencies

```bash
  pipenv install
```

Start the Bot

```bash
  pipenv run python -m bot
```
